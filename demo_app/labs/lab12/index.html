Лабораторная работа: Определение показателя преломления стекла линзы

Теоретическая часть

Показатель преломления стекла (n) можно определить, измерив параметры двояковыпуклой линзы и её фокусное расстояние.

Для двояковыпуклой линзы действует формула: n = (R/2F) + 1, где R - радиус кривизны сферических поверхностей линзы, F - фокусное расстояние.

Радиус кривизны (R) вычисляется по формуле: R = (D²/4 + (H-h₀)²)/(2*(H-h₀)), где D - диаметр линзы, H - толщина линзы в центре, h₀ - толщина линзы на краю.

Измерение параметров линзы

Используйте штангенциркуль для измерения толщины (H), диаметра (D) и толщины края (h₀) линзы. Перетащите экран для определения фокусного расстояния (F).
